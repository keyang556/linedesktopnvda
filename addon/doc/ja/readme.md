# LINE デスクトップ版 NVDA アドオン

## 概要

このアドオンは、Windows 版 LINE デスクトップアプリ（Qt6 バージョン）の NVDA スクリーンリーダーサポートを強化します。OCR（文字認識）と座標ベースの自動化技術を使用して、チャットリストの移動、メッセージの読み上げ、通話機能のアクセシビリティを向上させます。

> [!IMPORTANT]
> このアドオンは一部の機能に OCR（文字認識）を使用しているため、認識結果が完全に正確ではない場合があります。

## 特徴

* チャットリストやメッセージ入力欄のナビゲーションを改善。
* **音声・ビデオ通話**：友だちタブから通話を開始します。
* **着信対応**：着信の応答、拒否、発信者情報の確認が可能です。
* **OCR サポート**：標準的なアクセシビリティ API では取得できないテキストを自動的に認識しようとします。
* **デバッグツール**：UI 構造を確認し、トラブルシューティングに役立つショートカットを提供します。

## 使用上のヒントと注意点

* **接続の確立**：このアドオンを使用して初めてメッセージを送信する前に、一度スマートフォンまたは Chrome 版でメッセージを送信しておくことをお勧めします。チャット履歴があることで、アドオンが UI 要素を特定しやすくなります。
* **メッセージの送信**：
    1. 友達の名前を検索します。間違いを避けるため、検索結果が 1 つになるようなキーワードを使用してください。
    2. メッセージリストやサイドバーで `Shift+Tab` を押し、編集エリアに移動します。
    3. メッセージを入力して `Enter` キーを押すと送信されます。
* **電話をかける**：
    現在、このアドオンを使用してチャットタブから直接電話をかけることはできません。以下の手順に従ってください：
    1. `Control+1` を押して友だちタブに切り替えます。
    2. 友だちの名前を検索し、選択してチャットルームを開きます。
    3. `NVDA+Windows+C` で音声通話、または `NVDA+Windows+V` でビデオ通話を開始します。
* **確認**：メッセージの送信や通話を行う前に、必ずチャット履歴を確認し、送信先が正しいか確かめてください。

## メッセージの読み上げとコピー

* **メッセージの読み上げ**：メッセージリスト内を移動すると、アドオンは「コピー優先」方式でメッセージを読み上げます。具体的には、メッセージを自動で右クリック → 「コピー」を選択 → クリップボードの内容を読み上げます。読み上げ後、元のクリップボードの内容は自動的に復元されます。
* **メッセージのコピー**：メッセージリスト内で `Control+C` を押すと、アドオンが右クリックメニューの「コピー」機能を使ってメッセージをクリップボードにコピーします。
* **OCR フォールバック**：右クリックメニューからのコピーに失敗した場合（メニューやメニュー項目が見つからない場合など）、アドオンは自動的に OCR（文字認識）に切り替えてメッセージ内容を読み取ります。

### サウンドエフェクト

* **デフォルト（コピー成功）**：右クリックメニューのコピーでメッセージを読み取った場合、特別な音は鳴りません。
* **OCR フォールバック音**：OCR でメッセージを読み取った場合、`ocr.wav` のサウンドエフェクトが再生され、読み取った内容が完全に正確でない可能性があることを示します。

> [!WARNING]
> OCR（文字認識）の結果は**100% 正確ではありません**。`ocr.wav` の音が聞こえた場合、読み上げられたテキストが実際のメッセージ内容と異なる可能性があることにご注意ください。正確な内容を確認するには、スマートフォンまたは他のプラットフォームでメッセージを確認することをお勧めします。

## キーボードショートカット

> [!NOTE]
> 「カテゴリ」列の「アドオン」はこのアドオンが提供するショートカット、「LINE」は LINE デスクトップ版内蔵のショートカットを示します。

### 通話・着信

| ショートカット | カテゴリ | 操作 |
|---|---|---|
| **NVDA+Windows+C** | アドオン | 音声通話を開始 |
| **NVDA+Windows+V** | アドオン | ビデオ通話を開始 |
| **NVDA+Windows+A** | アドオン | 着信に応答 |
| **NVDA+Windows+D** | アドオン | 着信を拒否 |
| **NVDA+Windows+S** | アドオン | 発信者を確認 |
| **NVDA+Windows+F** | アドオン | 通話ウィンドウにフォーカス |
| **Ctrl+Shift+A** | LINE | マイクのオン／オフ |
| **Ctrl+Shift+V** | LINE | カメラのオン／オフ |

### メッセージ操作

| ショートカット | カテゴリ | 操作 |
|---|---|---|
| **Control+C** | アドオン | 現在のメッセージをコピー（メッセージリスト内で使用、編集欄では通常のコピー） |
| **NVDA+Windows+R** | アドオン | 現在のメッセージに返信 |
| **NVDA+Windows+Delete** | アドオン | 現在のメッセージを取り消し（送信取り消し） |
| **NVDA+Windows+T** | アドオン | 現在のチャットルーム名を読み上げ |
