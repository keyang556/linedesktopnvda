# LINE デスクトップ版 NVDA アドオン

## 概要

このアドオンは、Windows 版 LINE デスクトップアプリ（Qt6 バージョン）の NVDA スクリーンリーダーサポートを強化します。OCR（文字認識）と座標ベースの自動化技術を使用して、チャットリストの移動、メッセージの読み上げ、通話機能のアクセシビリティを向上させます。

> [!IMPORTANT]
> このアドオンは一部の機能に OCR（文字認識）を使用しているため、認識結果が完全に正確ではない場合があります。

## 特徴

* チャットリストやメッセージ入力欄のナビゲーションを改善。
* **音声・ビデオ通話**：チャット画面から直接通話を開始できます。
* **OCR サポート**：標準的なアクセシビリティ API では取得できないテキストを自動的に認識しようとします。
* **デバッグツール**：UI 構造を確認し、トラブルシューティングに役立つショートカットを提供します。

## 使用上のヒントと注意点

* **接続の確立**：このアドオンを使用して初めてメッセージを送信する前に、一度スマートフォンまたは Chrome 版でメッセージを送信しておくことをお勧めします。チャット履歴があることで、アドオンが UI 要素を特定しやすくなります。
* **メッセージの送信**：
    1. 友達の名前を検索します。間違いを避けるため、検索結果が 1 つになるようなキーワードを使用してください。
    2. メッセージリストやサイドバーで `Shift+Tab` を押し、編集エリアに移動します。
    3. メッセージを入力して `Enter` キーを押すと送信されます。
* **確認**：メッセージの送信や通話を行う前に、必ずチャット履歴を確認し、送信先が正しいか確かめてください。
* **通話の制限**：現在は発信のみをサポートしており、着信に応答することはできません。

## キーボードショートカット

| ショートカット | カテゴリ | 操作 |
|---|---|---|
| **NVDA+Shift+C** | LINE Desktop | 音声通話を開始 |
| **NVDA+Shift+V** | LINE Desktop | ビデオ通話を開始 |
| **NVDA+Shift+T** | LINE Desktop | 添付ファイル選択ウィンドウを開く |
| **NVDA+Shift+K** | LINE Desktop | デバッグ：UIA と OCR を確認（クリップボードにコピー） |
| **NVDA+Shift+J** | グローバル | フォーカスされているアプリとプロセス情報を報告 |

## 対応バージョン

* Windows 版 LINE デスクトップ (通常版または Microsoft Store 版)。
* NVDA 2022.1 以降。
