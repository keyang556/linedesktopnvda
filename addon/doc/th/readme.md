# ส่วนเสริม NVDA สำหรับเพิ่มประสิทธิภาพการเข้าถึง LINE Desktop

## ภาพรวม (Overview)

ส่วนเสริม (Add-on) นี้ช่วยเพิ่มประสิทธิภาพการทำงานของโปรแกรมอ่านหน้าจอ NVDA สำหรับแอปพลิเคชัน LINE บน Windows (เวอร์ชัน Qt6) โดยจะช่วยอำนวยความสะดวกในการนำทางห้องแชท การอ่านข้อความ และการโทรออกผ่านระบบ OCR และการสั่งการแบบระบุพิกัดอัตโนมัติ

> [!IMPORTANT]
> **ข้อควรทราบ**: ส่วนเสริมนี้มีการใช้เทคโนโลยี OCR (Optical Character Recognition) ในบางฟังก์ชัน ผลลัพธ์ที่ได้อาจไม่ถูกต้องแม่นยำ 100%

## คุณสมบัติเด่น (Features)

* **ปรับปรุงระบบการนำทาง**: สำหรับรายการแชทและช่องกรอกข้อความ
* **การโทรด้วยเสียงและวิดีโอ**: เริ่มการโทรจากแท็บเพื่อน
* **การจัดการสายเรียกเข้า**: รับสาย ปฏิเสธสาย หรือตรวจสอบข้อมูลผู้โทร
* **รองรับระบบ OCR**: พยายามอ่านข้อความที่ระบบการเข้าถึงมาตรฐาน (Accessibility APIs) เข้าไม่ถึงโดยอัตโนมัติ
* **เครื่องมือช่วยเหลือ (Debug Tools)**: มีคีย์ลัดสำหรับตรวจสอบโครงสร้าง UI เพื่อใช้ในการแก้ไขปัญหา

## คำแนะนำการใช้งาน (Usage Tips)

* **การสร้างการเชื่อมต่อ**: ก่อนจะใช้ส่วนเสริมนี้ส่งข้อความหาใครเป็นครั้งแรก แนะนำให้ส่งข้อความหาคนนั้นผ่านโทรศัพท์มือถือหรือ Chrome Extension ก่อนอย่างน้อยหนึ่งข้อความ เนื่องจากประวัติการแชทจะช่วยให้ส่วนเสริมค้นหาตำแหน่งขององค์ประกอบต่าง ๆ ได้แม่นยำยิ่งขึ้น
* **การส่งข้อความ**:
    1. ค้นหาชื่อเพื่อน โดยพยายามใช้คำค้นหาที่แสดงผลลัพธ์เพียงรายการเดียวเพื่อป้องกันความผิดพลาด
    2. เมื่ออยู่ในรายการแชทหรือแถบด้านข้าง ให้กด `Shift+Tab` เพื่อไปยังช่องพิมพ์ข้อความ (Edit field)
    3. พิมพ์ข้อความแล้วกด `Enter` เพื่อส่ง
* **การโทรออก**:
    ขณะนี้ไม่สามารถกดโทรออกได้โดยตรงจากแท็บแชทด้วยส่วนเสริมนี้ โปรดทำตามขั้นตอนต่อไปนี้:
    1. กด `Control+1` เพื่อสลับไปยังแท็บเพื่อน
    2. ค้นหาชื่อเพื่อน เลือกเพื่อน และเปิดห้องแชท
    3. กด `NVDA+Windows+C` เพื่อโทรด้วยเสียง หรือกด `NVDA+Windows+V` เพื่อโทรด้วยวิดีโอ
* **การตรวจสอบ**: ควรตรวจสอบประวัติการแชทและชื่อผู้รับให้แน่ใจทุกครั้งก่อนจะส่งข้อความหรือกดโทรออก

## การอ่านและคัดลอกข้อความ

* **การอ่านข้อความ**: เมื่อเลื่อนดูรายการข้อความ ส่วนเสริมจะใช้วิธี "คัดลอกก่อน" ในการอ่านข้อความ โดยจะคลิกขวาที่ข้อความโดยอัตโนมัติ → เลือก "คัดลอก" → อ่านเนื้อหาจากคลิปบอร์ด เนื้อหาคลิปบอร์ดเดิมจะถูกกู้คืนหลังจากอ่านเสร็จ
* **การคัดลอกข้อความ**: กด `Control+C` ในรายการข้อความ ส่วนเสริมจะคัดลอกข้อความไปยังคลิปบอร์ดผ่านเมนูคลิกขวา
* **การใช้ OCR สำรอง**: หากการคัดลอกผ่านเมนูคลิกขวาล้มเหลว (เช่น ไม่พบเมนูหรือรายการเมนู) ส่วนเสริมจะสลับไปใช้ OCR (การรู้จำอักขระด้วยแสง) เพื่ออ่านเนื้อหาข้อความโดยอัตโนมัติ

### เอฟเฟกต์เสียง

* **ค่าเริ่มต้น (คัดลอกสำเร็จ)**: หากข้อความถูกอ่านผ่านการคัดลอกจากเมนูคลิกขวา จะไม่มีเสียงเอฟเฟกต์พิเศษ
* **เสียง OCR สำรอง**: หากข้อความถูกอ่านผ่าน OCR จะมีเสียง `ocr.wav` ดังขึ้นเพื่อแจ้งว่าเนื้อหาที่อ่านได้อาจไม่ถูกต้องทั้งหมด

> [!WARNING]
> ผลลัพธ์จาก OCR (การรู้จำอักขระด้วยแสง) **ไม่ถูกต้อง 100%** หากคุณได้ยินเสียง `ocr.wav` โปรดทราบว่าข้อความที่อ่านได้อาจแตกต่างจากเนื้อหาจริง หากต้องการยืนยันเนื้อหาที่แน่นอน แนะนำให้ตรวจสอบข้อความบนโทรศัพท์มือถือหรือแพลตฟอร์มอื่น

## คีย์ลัด (Keyboard Shortcuts)

> [!NOTE]
> ในคอลัมน์ "หมวดหมู่" คำว่า "ส่วนเสริม" หมายถึงคีย์ลัดที่ส่วนเสริมนี้จัดเตรียมให้ และ "LINE" หมายถึงคีย์ลัดที่มาพร้อมกับ LINE Desktop

### การโทรและสายเรียกเข้า

| คีย์ลัด | หมวดหมู่ | คำสั่ง |
|---|---|---|
| **NVDA+Windows+C** | ส่วนเสริม | เริ่มการโทรด้วยเสียง (Voice Call) |
| **NVDA+Windows+V** | ส่วนเสริม | เริ่มการโทรด้วยวิดีโอ (Video Call) |
| **NVDA+Windows+A** | ส่วนเสริม | รับสายเรียกเข้า |
| **NVDA+Windows+D** | ส่วนเสริม | ปฏิเสธสายเรียกเข้า |
| **NVDA+Windows+S** | ส่วนเสริม | ตรวจสอบข้อมูลผู้โทร |
| **NVDA+Windows+F** | ส่วนเสริม | ไปยังหน้าต่างการโทร |
| **Ctrl+Shift+A** | LINE | เปิด/ปิดไมโครโฟน |
| **Ctrl+Shift+V** | LINE | เปิด/ปิดกล้อง |

### การจัดการข้อความ

| คีย์ลัด | หมวดหมู่ | คำสั่ง |
|---|---|---|
| **Control+C** | ส่วนเสริม | คัดลอกข้อความปัจจุบัน (ใช้ในรายการข้อความ; ในช่องแก้ไขจะเป็นการคัดลอกปกติ) |
| **NVDA+Windows+R** | ส่วนเสริม | ตอบกลับข้อความปัจจุบัน |
| **NVDA+Windows+Delete** | ส่วนเสริม | เรียกคืนข้อความ (ยกเลิกการส่ง) |
| **NVDA+Windows+T** | ส่วนเสริม | อ่านชื่อห้องแชทปัจจุบัน |
