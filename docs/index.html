LINE NVDA 桌面版附加元件

開發動機

2025年8月12日，LINE 宣布將於同年9月22日結束 Chrome 擴充功能版本。這對視障使用者來說是巨大的衝擊，因為 Chrome 版是目前視障使用者唯一可以透過電腦操作、且相對友善（儘管仍不完美）的版本。後續雖因引發爭議而暫停下架，但該版本仍隨時面臨終止的風險。為了確保數位平權與通訊暢通，我們開發了此附加元件，致力於全面提升 LINE 官方桌面版軟體的無障礙操作體驗。

核心功能

  • 修復訊息列表的鍵盤焦點與流暢的報讀順序。
  • 優化聯絡人清單與聊天室之間的快速切換（Tab 鍵導航）。
  • 支援未讀訊息數量與來電提示的即時語音報讀（aria-live）。
  • 修正部分無法透過鍵盤觸發的按鈕元件。

系統需求

  • 螢幕閱讀器：NVDA 2019.3 或以上版本（建議使用 2024.1 或更新版本）。
  • 通訊軟體：LINE 電腦版 for Windows（從 Microsoft Store 安裝的最新版本）。
  • 作業系統：Windows 10 / Windows 11。

下載與安裝

請前往專案的 GitHub Releases 頁面下載最新版本的 `.nvda-addon` 檔案。下載後，在執行 NVDA 的狀態下，直接開啟該檔案即可完成安裝。
