## 注意事項

- 所有快速鍵組合（如 `NVDA+Windows+K`、`NVDA+Windows+P`）在四個語言版本中必須保持完全一致
- `NVDA+Windows+K` 在兩個不同的表格中出現：
  - **訊息操作表格**：開啟另存新檔對話框（用於檔案訊息，包含 7 天備註）
  - **媒體播放器表格**：儲存語音訊息（用於語音訊息，無需備註）
- 新增內容必須遵循現有文檔的 Markdown 格式規範
- 虛擬視窗章節的位置必須在「訊息閱讀與複製」之後、「鍵盤快速鍵」之前
- 音效提示內容移動時必須保持完整，不可遺漏任何說明
- 各語言版本的章節結構和順序必須保持一致
